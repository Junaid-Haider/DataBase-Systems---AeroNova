# Axiom Airlines Management System - Frontend

This is the frontend application of the **Axiom Airlines Management System**, built using React, Vite, TypeScript, and Tailwind CSS.

For full project details, setup guides, architecture descriptions, and backend guidelines, please refer to the **[Main README in the root directory](../README.md)**.

## 🛠️ Tech Stack & Key Libraries
*   **Core:** React 18 + Vite + TypeScript
*   **Styling:** Tailwind CSS + PostCSS
*   **State Management:** Zustand
*   **Data Fetching:** TanStack React Query + Axios
*   **Animations:** Framer Motion + Three.js (Globe effect)
*   **Charts:** Recharts
*   **Icons:** Lucide Icons

## 🚀 Running Locally

1.  Make sure dependencies are installed:
    ```bash
    npm install
    ```
2.  Set up your environment variables:
    Create a `.env` file (or copy `.env.example`):
    ```env
    VITE_API_URL=http://localhost:3001/api/v1
    ```
3.  Start the local development server:
    ```bash
    npm run dev
    ```
4.  Build for production:
    ```bash
    npm run build
    ```
